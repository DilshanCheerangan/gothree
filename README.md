# GoThree | Next-Gen Luxury Landing Page

A high-performance, cinematic internship landing page built for **GoThree**. This project features a sophisticated dark-mode aesthetic with liquid glass design, smooth scroll integration, and a native registration system connected to Google Forms.

## 🚀 Features

- **Premium UI**: Custom "Liquid Glass" design system using vanilla CSS and Tailwind.
- **Cinematic Animations**: Scroll-linked parallax and entrance animations powered by Framer Motion and GSAP.
- **Native Registration**: High-performance form mapping directly to Google Forms with a seamless success state.
- **Smooth Navigation**: Optimized anchor links with a smart floating "pill" Navbar that adapts to scroll depth.
- **SEO Optimized**: Includes meta tags, `robots.txt`, and clean semantic HTML.

## 🛠️ Tech Stack

- **Framework**: Next.js 16 (App Router)
- **Styling**: Tailwind CSS v4
- **Animation**: Framer Motion, GSAP, Lenis (Smooth Scroll)
- **Icons**: Lucide React

## 📦 Getting Started

1. **Install Dependencies**:
   ```bash
   npm install
   ```

2. **Run Local Development**:
   ```bash
   npm run dev
   ```

3. **Build for Production**:
   ```bash
   npm run build
   ```
